# web-api-security-study
## 📌 목표
Web API의 보안 메커니즘을 체계적으로 학습하고,
공격자 관점에서 잠재적인 공격 표면(Attack Surface)을 분석하는 것을 목표로 한다.

## 🧠 학습 범위
- HTTP 프로토콜 심층 분석
- REST 아키텍처 보안 구조
- 인증(Authentication) 및 인가(Authorization) 모델
- JWT 및 토큰 기반 인증 시스템
- OWASP API Security Top 10 분석
- 공격 흐름(Attack Flow) 모델링

## 📂 구조
theory/      → 핵심 개념 및 프로토콜 보안 분석  
analysis/    → 취약점 모델링 및 공격 흐름 분석  

## 🚀 최종 목표
API 취약점을 식별하고, 공격 시나리오를 설계하며,
이를 효과적으로 방어할 수 있는 역량을 기르는 것.
